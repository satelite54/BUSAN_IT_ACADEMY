Git 개발환경 셋팅

1. 사용자 정보

git config --global user.name "영문이름"
git config --global user.email "메일주소@example.com"

2. 초기화

git init

3. 파일 생성
touch 파일이름
touch 파일이름 파일이름 파일이름

4. Git 상태확인
git status

5. git add .
.은 모두 추가

6. git commit -m "파일이름"
git commit -m . 전체커밋
